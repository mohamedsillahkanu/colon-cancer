Implementation Methodology

1 Data Collection: Gathered 25,000 histopathological images across 5 classes with organized folder structure.
2 Preprocessing: Resized images to 150×150 pixels and normalized pixel values for optimal performance.
3 Data Augmentation: Applied rotation, width/height shifts, shear, zoom, and horizontal flips to enhance dataset diversity.
4 Model Design: Implemented CNN with multiple convolutional layers, pooling layers, and dense layers.
5 Training: Used 80/20 train-test split with Keras validation_split during training, categorical crossentropy loss and Adam optimizer.
6 Evaluation: Assessed performance using accuracy and loss metrics, achieving 93.3% test accuracy.

Model Architecture:

model = Sequential([ Conv2D(32, (3, 3), activation='relu', input_shape=(150, 150, 3)), MaxPooling2D(pool_size=(2, 2)), Conv2D(64, (3, 3), activation='relu'), MaxPooling2D(pool_size=(2, 2)), Flatten(), Dense(128, activation='relu'), Dropout(0.5), Dense(5, activation='softmax') ])
